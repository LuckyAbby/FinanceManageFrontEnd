## 小型的理财系统前端界面

### Usage
```
git clone https://github.com/LuckyAbby/FinanceManageFrontEnd.git

cd FinanceManageFrontEnd

yarn

yarn start
```

### 技术栈
使用 Ant Design + Umi + Dva

### 功能模块
- 产品介绍
- 产品购买
- 产品赎回
- 产品卖出
- 个人资产
- 交易详情
- 个人信息

### 配置信息
- 页面 Layout 采用基本的 BasicLayout 布局
- 路由配置位于 `.umirc.js` 文件中
